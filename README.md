# FNFLC

Hello! This project is not affiliated with Newgrounds or Funkin' Crew Inc. It is a completely fan-made, non-profit project.

![FNFLC](data/logo.gif)

#### ![FNFLC](data/big-icon.gif) Friday Night Funkin' Legacy Console (FNFLC) is a fan-made project dedicated to porting Friday Night Funkin' to various retro gaming consoles, ranging from 8-bit to 32-bit systems. ![FNFLC](data/big-icon.gif)

## Supported Platforms:
### Handhelds
* **8-Bit:** None
* **16-Bit:** None
* **32-Bit:** Nintendo Game Boy Advance, Nintendo DS
